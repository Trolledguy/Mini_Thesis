Setting For Project Ediotor

Unity Editor Version : 6000.3.4f1

!!! ข้อบังคับ !!!

- ห้ามทำงานในไฟล์ Unity Scene เดียวกันเด็ดขาด !!!
- เมื่อ Clone และ pull งานไปแล้วแนะนำให้แตกหน่อ branch ออกไปทำงานในส่วนของตัวเอง
  - คำสั่ง
    - git branch     =>  List ชื่อ branch ทั้งหมดใน Repo
    - git branch (ชื่อ) => สร้าง branch พร้อมชื่อ
    - git switch (ชื่อ) => เปลี่ยน branch
- เมื่อทำงานเสร็จแล้วอย่าลืม commit แล้ว push นะจ้ะจุ้บๆ
    - git add .    =>   เพิ่มไฟล์ทั้งหมดมาเตรียม commit
    - git commit -m "ข้อความ" => commit ไฟล์
    - git push      =>  อัพโหลดขึ้น github ให้คนอื่นๆเห็น
